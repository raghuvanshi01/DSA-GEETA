# DSA Bible

> Learn algorithms by recognizing patterns, not by memorizing solutions.

The DSA Bible is a comprehensive interview preparation resource that teaches:

- Pattern Recognition
- Algorithm Intuition
- Java Templates
- Proofs
- Complexity Analysis
- Interview Thinking
- Curated LeetCode Problems
- Company-wise Preparation
- Interactive Visualizations

The goal is simple:

> Given any unseen DSA problem, identify the correct pattern within 30 seconds.
